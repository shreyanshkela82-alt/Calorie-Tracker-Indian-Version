# Indian Calorie Tracker

A modern web-based **Indian Calorie Tracker** project developed by **Shriyansh Kela**.  
This project helps users track daily calories, manage Indian meals, calculate BMI, set fitness goals, and get workout suggestions based on their calorie intake.

## Project Description

Indian Calorie Tracker is a fitness and nutrition tracking web application designed especially for Indian users. It allows users to enter their profile details such as weight, height, age, and fitness goal. Based on this information, the app calculates BMI and daily calorie requirements.

Users can add Indian food items like roti, dal, rice, sabji, poha, idli, dosa, and other common meals. The app calculates the total calories consumed and compares it with the user’s daily calorie target. It also provides workout suggestions when calorie intake goes above the required limit.

## Features

- User profile management

- Fitness goal selection

## How It Works

1. User creates an account or logs in.
2. User updates profile details such as height, weight, age, and goal.
3. App calculates BMI and daily calorie requirement.
4. User adds meals eaten during the day.
5. App calculates consumed calories.
6. App shows remaining calories and progress.
7. If extra calories are consumed, workout suggestions are shown.

## Project Purpose

The main purpose of this project is to help users maintain a healthy lifestyle by tracking their daily food intake and fitness goals. It is also built as a portfolio project to demonstrate frontend development skills, DOM manipulation, local storage usage, and UI design.
